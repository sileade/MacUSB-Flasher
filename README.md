# MacUSB Flasher

**MacUSB Flasher** — это универсальная консольная утилита для macOS (включая Apple Silicon M1/M2/M3/M4), предназначенная для создания загрузочных USB-накопителей.

Утилита автоматизирует процесс форматирования, разбивки и записи образов, избавляя от необходимости вручную вводить сложные команды в терминале.

## 🌟 Возможности

| Режим | Метод записи | Поддерживаемые форматы |
|-------|-------------|------------------------|
| **Windows 10/11 (UEFI)** | FAT32 + GPT + wimlib split | `.iso` |
| **Linux** (Ubuntu, Debian, Mint, Arch...) | `dd` на `/dev/rdisk` | `.iso` |
| **Raspberry Pi / ARM** (Raspbian, DietPi...) | `dd` на `/dev/rdisk` | `.img`, `.img.gz`, `.img.xz`, `.zip` |
| **Универсальный режим** | `dd` на `/dev/rdisk` | любой `.iso` / `.img` |

### Ключевые особенности:
- **Автоопределение флешки:** Скрипт сам найдет новую флешку при подключении (защита от случайного форматирования системного диска).
- **Автоустановка зависимостей:** Самостоятельно установит `Homebrew`, `wimlib` и `xz` при необходимости.
- **Поддержка больших файлов Windows:** Автоматически разбивает файл `install.wim` (если он больше 4 ГБ) для совместимости с файловой системой FAT32.
- **Распаковка на лету:** Поддерживает прямую запись сжатых образов (`.gz`, `.xz`, `.zip`).
- **Максимальная скорость:** Использует `/dev/rdisk` для ускорения побитовой записи.

## 🚀 Установка

Вы можете установить утилиту на рабочий стол для удобного запуска двойным кликом.

1. Скачайте репозиторий:
```bash
git clone https://github.com/sileade/MacUSB-Flasher.git
cd MacUSB-Flasher
```

2. Запустите установщик:
```bash
chmod +x install_macusb_flasher.sh
./install_macusb_flasher.sh
```

На рабочем столе появится папка `MacUSB_Flasher`. Для запуска просто дважды кликните по файлу `MacUSB_Flasher.command`.

## 💻 Использование из терминала

Если вы не хотите устанавливать утилиту на рабочий стол, её можно запускать напрямую:

```bash
chmod +x MacUSB_Flasher.sh
./MacUSB_Flasher.sh
```

> **⚠️ ВАЖНО:** Не запускайте скрипт через `sudo`! Скрипт сам запросит пароль администратора только в тот момент, когда это потребуется для форматирования диска.

## 🛠 Как это работает (под капотом)

### Для Windows
macOS не поддерживает создание загрузочных флешек Windows через `dd`. Поэтому скрипт:
1. Форматирует флешку в `MS-DOS (FAT32)` со схемой разделов `GPT`.
2. Монтирует ISO-образ.
3. Копирует все файлы через `rsync`.
4. Если файл `sources/install.wim` превышает лимит FAT32 (4 ГБ), скрипт использует `wimlib-imagex` для его разделения на части (`.swm`).

### Для Linux и ARM
Используется классическая побитовая утилита `dd`. Для ускорения процесса запись идет на raw-устройство (`/dev/rdiskN` вместо `/dev/diskN`), а размер блока установлен в `4m`.
